> Image File을 어떻게 입출력할까?

```
Spring / input : (json(byte[]) / url 등) > Byte[] > (auto) > MediumBlob
Spring / output : MediumBlob > (auto) > Byte[] > json
Vue / input : json > (axios사용) "data:image/png;base64,"+ byte[] > `img`태그의 `src`에 활용
```
